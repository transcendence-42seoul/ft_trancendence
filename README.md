</br>

# 실행방법

.env_template 내용을 복사해서 복사해서 같은 위치에 .env파일을 생성하고 붙여넣기한다.

docker가 설치되어있다고 가정 후,

```
make
```
makefile을 실행 후, localhost:5173으로 접속한다.
</br>
</br>

# 프로젝트 기능


- 채팅 룸 생성
- 채팅 ban 기능
- 친구 추가
- 실시간 Pong 게임 with Socket
- 게임 중 실시간 채팅
- 42Seoul API, JWT를 이용한 로그인
